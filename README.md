# Django React Ecommerce Platform

A full-stack ecommerce platform built with Django REST Framework and React.

## Features

- JWT Authentication
- Admin Dashboard
- Product Management
- Categories
- Shopping Cart
- Wishlist
- Checkout
- Reviews
- User Dashboard

## Tech Stack

- Python
- React
- Django REST Framework
- PostgreSQL
- JWT Authentication
- Styled Components
- Framer Motion

## Running Locally

Backend:

```bash
python manage.py runserver
```

Frontend:
```bash
npm start
```
